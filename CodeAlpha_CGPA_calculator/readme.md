# CGPA Calculator Using C Programming

## Project Overview

This project is developed as part of the CodeAlpha C Programming Internship. The project focuses on creating a CGPA Calculator that calculates a student's academic performance based on subject grades and credit points.

The main objective of this project is to apply C programming concepts such as variables, loops, arrays, functions, and conditional statements to develop a useful academic calculation application.

## About CGPA Calculator

A CGPA Calculator is a program that calculates the overall Cumulative Grade Point Average (CGPA) of a student based on the grades obtained in different subjects.

The calculator uses subject grade points and credit hours to calculate the weighted average score, helping students evaluate their academic performance.

## Working of the Program

The CGPA Calculator works using the following steps:

1. User enters the number of subjects.
2. User provides grade points and credit hours for each subject.
3. The program calculates the total weighted grade points.
4. Total credit hours are calculated.
5. CGPA is calculated using the required formula.
6. Final CGPA is displayed to the user.

## Formula Used

CGPA = Total (Grade Point × Credit) / Total Credit
